Publication Schedule

GMM publishes every Monday at 08:00 UTC.

Each issue covers the seven-day period ending on the Sunday immediately before publication. The data window closes at approximately 06:00 UTC Monday to allow scoring and publication. The HTML shell is static and maintained separately from the data JSON files. Cron tasks update data only — never HTML.
